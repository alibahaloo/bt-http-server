Command Center

Enter the command in the box below and click on "Send" button.

For START command; Type START in the textbox followed by X-Coordinate ( In 5 Digits) and Y-Coordinate (In 5 Digits),Seperated by a space character:e.g: START XXXXX YYYYY. and Click on "Send" Button.

example of START command: START 00500 11611

For STOP Command, Type STOP in the textbox and click on "Send"  Button.
